IDEAS DE MOCKUPS PARA UNA POSIBLE APLICACIÓN
PROJECT TITLE: Gestión de lista de actividades 
PURPOSE OF PROJECT: Se propone un proyecto con el cual usted va a poder 
administrar listas de tareas o actividades y gestionarlas.
VERSION or DATE: 21/6/2021
HOW TO START THIS PROJECT: Para empezar este proyecto debe ejecutar el main
de la herramienta de gestión

USER INSTRUCTIONS: 
Para llevar a cabo este proyecto hemos implimentado un total
de nueve clases. Las cuales van a ser manejadas por la clase "Herramienta de gestión". 
Al ejecutar el main de esta clase el proyecto dictara opciones para dar inicio a 
un tipo de agenda de actividades. La cual es capaz de:
-Crear listas, para poder clasificar las actividades y tareas. Esto es un metodo
de este proyecto. Cada lista tendrá un identificador, un nombre y ,opcionalmente,
una descripción.
-A cada lista se le pueden agregar distintas tareas o actividades, las cuales requeriran
que el usuario ingrese un id, una fecha de inicio, una fecha de finalización,
un responsable, una estimación de recursos que requiera la tarea y un grado de 
avance, esto para agregar la información y la tarea a su respectiva lista.
-Estas listas y tareas se pueden modificar, y cambiar cada uno de los contenidos
de las variables.
-Además usted podrá asociar las tareas según su dependencia.
-A cada tarea se le asignara uno o varios colaboradores.
-Para el buen funcionamiento de esa parte, se decidio que lo mejor
era crear un banco de colaboradores, a los cuales se les asigna una tarea y 
el tiempo semanal que utilizará.
-Cada responsable, estado, avance, tarea y lista, se puede insertar, modificar,
excluir y consultar.
Este proyecto podría ser bastante util para una empresa. Donde los administradores
podrían agregar y asignar todas las tareas a cada empleado, además así 
tendrían un reporte de las actividades de una manera más eficiente. 
Ya que con solo seleccionar consultar, podría revisar el estado de cada actividad,
y así juzgar el desempeño de cada trabajador.
